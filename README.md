# paydash

### Language
 - NodeJS
 - ExpressJS
____

### Libraries
 - DataTables
 - pg-promise
 - tether.io
 - bootstrap

### Pre-Requisites
 + NodeJS
    - download from [https://nodejs.org](https://nodejs.org)
 + NPM
    - comes with NodeJS

___

### Running The Project
+ Clone project from ```this``` repo using:
    - ``` git clone https://github.com/emmanuelstroem/paydash.git```

+ cd into the project:
    - ```cd paydash```

+ Install dependencies using:
    - ``` npm install ```

+ Run the project using:
    - ``` npm start```
